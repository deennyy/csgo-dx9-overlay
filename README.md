# csgo-dx9-overlay

An external overlay using DX9 and ImGui for CS:GO, written for Windows in C++

![Screenshot](screenshot.png)

Cheat features: <br>
Box ESP <br>
You could pretty much add any other ESP function though, the box is just an example
